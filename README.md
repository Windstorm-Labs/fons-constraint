# The Fons Constraint

**The Fons Constraint: Information-Theoretic Convergence on Encoding Depth in Self-Replicating Systems**

**Paper 1 in the FONS Constraint / Throughput Basin series**

---

## Quick Summary

Derives the 64-codon genetic alphabet from first principles using Shannon information theory and Eigen's error threshold. Shows that stable molecular self-replication requires M < (1/ε)·ln(1/ε) — predicting the universal genetic code at observed error rates.

**Key Finding:** The 64-codon alphabet is information-theoretically optimal for molecular self-replication. Stable across 9+ orders of magnitude of error rate.

## About This Series

The FONS (Fons Optimal Numerus Signorum) constraint and Throughput Basin papers investigate whether serial decoding systems — from ribosomes to language models — converge to fundamental information-processing limits. The series combines theoretical derivation, computational experiment, and cross-substrate comparison.
